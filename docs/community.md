## 💡 提问技巧

在社区提问时，建议包含以下信息：

### ✅ 好的提问示例
```
环境信息：
- 操作系统：Windows 11
- Node.js 版本：22.0.0
- OpenClaw 版本：1.0.0

问题描述：
配置企业微信后，发送消息失败，提示 "权限不足"

已尝试的方法：
1. 检查了 appKey 和 appSecret 配置
2. 确认了服务器 IP 在白名单中
3. 重启了 OpenClaw 网关

日志信息：
[2024-01-01 10:00:00] ERROR: Permission denied for user xxx
```

### ❌ 不好的提问示例
```
企业微信配置失败了，怎么办？
```
